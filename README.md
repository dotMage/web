# web

dotMage Web Admin — read-only admin panel for the E2E-encrypted .env secret manager. React + TypeScript + Vite. See dotmage-spec for full specification.

**Important:** This admin panel does NOT display secret values — the server never has access to plaintext secrets (E2E encryption).

## Contributing

Every user-visible change updates `CHANGELOG.md` under `[Unreleased]` in the same PR —
entries are written for users, not committers.
